# Example House Rules

These are example local house rules. Copy this file to `house_rules.md` for private local use.

## Checks

- Only call for a roll when failure would be interesting.
- Failed investigation checks should not reveal success-only hidden information.
- Failed perception checks may introduce ambiguity, delay, noise, or risk.

## Tone

- Prefer grounded, low-level consequences over world-ending escalation.
- Keep scenes focused and interactive.
